## How to Start? ✨

```
# Step 0 : 처음 시작한 경우
yarn install

# Step 1 : 실행
yarn start
```

### commit message 🍡

| 헤더     | 내용                                                                |
| -------- | ------------------------------------------------------------------- |
| feature  | 새로운 기능 추가                                                    |
| refactor | 코드 리팩토링                                                       |
| fix      | 버그 수정                                                           |
| style    | 세미 콜론 수정, 줄바꿈, 스타일 등 기능에 지장 가지 않는 스타일 변경 |
| setting  | 빌드수행, 패키지 설치, 환경 설정 수정 등                            |
| document | 주석 추가삭제, reademe 작성                                         |
